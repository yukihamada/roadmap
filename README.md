# README　

## 目的
みんなが長期のビジョンを共有することで、長期的な視野で自分の仕事の本質的な価値を考え、
先を見通せるという心理的安全性を確保した状態で、業務に邁進できる組織を目指す。

## 原則
HD CPOロードマップはCPOが各チームと議論した10年後までの「目指す姿」をおとしたものになっています。
議論は常に行われ、ロードマップは都度アップデートされます。
HD CPOロードマップを起点に、ディビジョン/チーム/個人でロードマップを作成します。
各ロードマップはメルカリの組織の自らのチーム上にリンクをはってください。ファイルの保存先等詳細については下記「記載マニュアル」を参照してください。
各ロードマップはHD CPOロードマップ実現と平仄をとる形で、各自の「目指す姿」を記載したものになっています。
各自の「目指す姿」から逆算する形で、より近い未来のアクションプランを考え記載します。
結果として、各Div/各チーム/個人の四半期OKRだけでなく、数年後に至るOKRが常に更新された状態を保ち、将来の可視性を高める「新しい地図」としての役割を持ちます。

## CPOからの約束
CPOはHD CPOロードマップ実現にフォーカスした経営を行います。
環境変化や競争力等に鑑みHD CPOロードマップに変更は常に行われるものとします。
「いつのまにか変わっていた」状態を避けるため、変更時には速やかに通知します。
HD CPOロードマップの変更により各自が玉突き式に右往左往する状態を避けるため、半年未満の短期のビジョンは原則として変更を加えず、常に近い未来が可視的である状況をつくります。
但し、どうしても回避不可能に発生した事象でみなさんに至急のお願いをすることもあると思いますが、その場合はAll For Oneで協力しあいましょう。

## 皆さんへのお願い
ロードマップはトップダウンのみで完成されるべきものではありません。
各チームや個人のロードマップは、上長との1on1等で共有し、上長やチームのロードマップに不足していると思うパーツは積極的に指摘・議論してください。
上長は、その議論を通じて得た優れた視点は吸い上げて、自らがオーナーシップを持つロードマップに反映してください。
これはまだまだ完全なものではなく、ぜひ改善案などください

# ロードマップの記載マニュアル

## はじめに
ロードマップフォルダにアクセスし、新規のDocsファイルを作成します。フォルダはTeamsの順序に従い作成しますが、日本語(ja)と英語(en)各フォルダに保存してください。例えば、CTOであればCPOフォルダの下にファイルを作成します（Roadmap>ja>HD>CPO>HD_CTOロードマップ）。CTO配下の各チームのロードマップはCPOフォルダの下にCTOフォルダを作成し、CXOフォルダの下に新規Docsを作成します（例：Roadmap>ja>HD>CPO>CTO>AIロードマップ）。
ファイル名は以下のように設定してください。：(HD|JP_Mercari|JP_Merpay)_TEAMNAMEロードマップ
Docsファイルに、テンプレートをコピーします。ロードマップ記載の手引きは、読んだら消します。
ロードマップの冒頭にチーム名とオーナーを記入してください。
また、英語版（英語版の場合日本語版）へのリンクもオーナーの横につけておいてください（翻訳の仕組みの詳細はこちらを御覧ください）。
ロードマップは、メルカリ全社員がみるものなので、特に以下を意識してください。
閲覧権限を全社に付与しましょう
編集権限は自身、チームメンバー等絞って構いません
みんなに分かりやすい言葉で書いて下さい

## 書いてみましょう
全社/各カンパニー/各ディビジョンのロードマップをよんでみましょう（参照：メルカリの組織）。
上長はロードマップをチーム内で説明したり、議論しながら、理解の促進に努めて下さい。
全社/各カンパニー/各ディビジョンのロードマップが実現されているために、各時点で自分やチームがあるべき姿を自分のロードマップに書き込んでいきましょう。
そして、その各時点での「あるべき姿」を実現するために、そこから遡って、各時点でのあるべき状態やアクションプランを考え、ロードマップに埋めてみましょう。
例えば、2030年の上位レイヤーの目指す姿が「各国の首脳をおさえ、世界平和を実現する」であった場合のある人の思考例です。
2030年の目指す姿として上長が「各国の首脳をおさえ、世界平和を実現」と記載。
それを見て、自身の2030年の「ありたい状態」として「日本の総理大臣となる」と設定。
そのために、何をするべきかを逆算してアクションプランを検討。
2025年までに「各国と台頭に議論ができる素地を作る。史上最も愛され、信頼される外務相となる。」　などが記載されます
ここからは少しずつアクションプランを具体化されていきます。
2024年までに「外務相に着任。難しい交渉をまとめ、各国からの信頼を集める」
2022年までに「国会議員となり大きな派閥に入る」
2021年6月までに「国会への転身を決意。出馬する選挙区を決める」
2020年までに「神奈川県会議員となる」　…といった具合です
このロードマップは各自/各チームのロードマップですので、すべてが全社/各カンパニー/各ディビジョンのロードマップと紐付いている必要はありません。自分がメルカリで目指したい姿は何でも記載し、アクションプランを埋めていきましょう。

## 議論しましょう
記載したロードマップは上長と議論しましょう。
上長と納得できるまで議論しましょう。その結果、目指す状態やアクションプランが変わったら、それを反映しましょう。
また、他チームからのサポートが必要なアクションや、上位のロードマップに欠けている視点がある場合には、積極的に上長にインプットして下さい。
上長は、重要度に応じてそれらの視点は吸い上げて、チームのロードマップに反映するとともに、更に上位のロードマップに反映することを検討し、アクションに移してください。

## 書き続けましょう
ロードマップは自分が手にした情報や環境によって日夜変わっていくものだと思います。
自らの目標とするとともに、未来日記として、定期的に考え、更新していきましょう。
何かを優先させるために、思ったタイミングで何かが実行できないこともあるかもしれません。その場合でも、そのアクションを近い将来実行する必要があるのであれば、「いつまでにやる必要があるか」「いつまでなら出来るのか」を考えながら、更新しましょう。
優先順位付けに悩んだら、そのときはまた上長と相談してみましょう。

## やり遂げたときは
記載したことをやり遂げたときは、文字を打ち消し線で消して下さい。そうすることで、誰もがあなたやチームに問い合わせることなく、それが終わったことが分かります。
そうすることで、それが終わったことを前提に他のチームも動け、よりスムースに組織をスケールする一助となります。

## 書き方のポイント
本ロードマップの目的は「みんなが長期のビジョンを共有することで、長期的な視野で自分の仕事の本質的な価値を考え、先を見通せるという心理的安全性を確保した状態で、業務に邁進できる組織を目指す。」ことです。
特に遠い将来については、具体的な打ち手や施策ではなく、目指す姿・状態を書くように心がけて下さい。技術革新や環境変化により、現在考えている施策は施策でなくなる場合がありますが、みなさんが考える「目指す状態」は、かけがえのないもので、施策を問わず目指すべきものだと思います。
一方で、半年以内の将来について、手段が記載可能なところは是非記載してください。それがOKRに繋がる重要なKey Resultsです。
